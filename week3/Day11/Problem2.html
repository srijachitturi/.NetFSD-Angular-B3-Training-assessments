Frequently Asked Questions

1. What is jQuery?

jQuery is a JavaScript library.

2. Why do we use jQuery?

It simplifies DOM manipulation and event handling.

3. What is $(document).ready()?

It ensures the DOM is fully loaded before executing code.

4. What is slideToggle()?

It shows or hides content with sliding animation.

5. What are jQuery selectors?

Selectors are used to select and manipulate HTML elements.
